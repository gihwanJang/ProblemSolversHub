### 생각의 흐름
간단한 조건문과 구현으로 처리하였습니다.<br>
이 때 수학이 어느정도 쓰였습니다.

### 수학
1. 키패드의 키 사이 거리 계산
- 3개씩 있으므로 수학적 계산 이용 (배열을 따로 만들지 않고)
- 3으로 나눈 몫(정수), 3으로 나눈 나머지 사이 거리 비교
- 이 때 거리는 유클리드 X, 맨해튼 거리 사용

2. 맨해튼 거리 사용 이유
- 유클리드 거리 사용 : 2번-0번 (거리 3), 6번-0번 (거리 2.xx)<br>
- 맨해튼 거리 사용 : 2번-0번 (거리 3), 6번-0번 (거리 3) => 주손 판단<br>
